# Design Context

## Target Audience
Developers and engineers — people who want a capable unit/expression calculator without fuss.

## Use Cases
Quick unit conversions, arithmetic with units, defining variables in a session, referencing results for follow-up calculations.

## Brand Personality
**Three words:** Mineral. Geological. Precise.

Warm creams, stone greys, earthy browns. Solid and trustworthy, like carved rock. Calm precision — no flash, all substance.

## Tone
Technical utility tool. No decorative flair. The aesthetic should feel like something built to last — the kind of tool you keep because it just works, not because it looks exciting.

## Reference Colors (from favicon.svg)
- Top face: `#f6f1e6` — warm cream/linen (`oklch(95% 0.025 80)`)
- Right face: `#d9cfb8` — warm tan/wheat (`oklch(85% 0.030 75)`)
- Shadow face: `#7a7466` — warm brown/slate (`oklch(52% 0.025 70)`)

## Theme
System default — follow `prefers-color-scheme`. Light: warm linen surfaces. Dark: deep granite with warm tints.

## Anti-references
Nothing that looks like a generic SaaS product. No blue accent palettes. No purple gradients. No glowing neon on dark. No cards-everywhere layout. Should feel like handmade precision instruments, not a startup app.
